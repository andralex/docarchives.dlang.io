    Module core.stdc.inttypes

    D header file for C99.

    This module contains bindings to selected types and functions from the standard C header &lt;inttypes.h&gt;. Note that this is not automatically generated, and may omit some types/functions from the original C header.

    Standards

    ISO/IEC 9899:1999 (E)
